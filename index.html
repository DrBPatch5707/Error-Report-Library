This library aims to provide consistent documentation for errors through out a project.
below are some helpful directions, you don't have to follow them, use the library however you need to.

Directions:

-use the namespace err

-create a errorReport object

-(optional but recomended) provide an output file through the setErrorFile() function (required to use the writeErrors() function)

-use the setError() function to describe an error.

-if you're doing something sketchy I would recomend useing

try {
//sketchy code
}
catch(/*other catches*/)
{setError("error for catch");}
catch(...)
{ writeErrors();}

to make sure the errors are recorded to the file

-if you need the errors in order printErrors() will print the errors to the console in order
